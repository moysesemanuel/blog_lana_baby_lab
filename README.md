# Lana Baby Lab

Blog de receitas para bebês com páginas públicas, Studio privado para cadastro de receitas e autenticação da administradora.

## Rodando localmente

```bash
yarn
yarn dev
```

App padrão em `http://localhost:3004`.

## Variáveis de ambiente

Use o arquivo `.env` com base em `.env.example`.

## Estrutura

- `server.js`: servidor Node
- `app/`: componentes, cliente, auth e dados
- `content/`: conteúdo persistido localmente
- `styles.css`: estilos globais
